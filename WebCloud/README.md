 # Дипломный проект - облачное хранилище

RESTfull приложение для облачного хранения файлов с авторизацией пользователя

## Стек технологий

*   **Язык:** Java 17
*   **Фреймворк:** Spring Boot  3.5
*   **Сборка:** Maven 
*   **Инфраструктура:** Docker
*   **База данных:** H2 Database


## Функциональность

* Регистрация и авторизация пользователей
* Загрузка файлов на сервер
* Скачивание файлов
* Удаление файлов
* Переименование файлов
* Получение актуального списка файлов пользователя
* Docker контейнеризация

## API Endpoints

### Авторизация
|Метод|	URL	           |Описание	     |Тело запроса                      | 
|-----|----------------|-----------------|----------------------------------|
|POST |	/cloud/login   |Вход в систему	 |{"login":"admin","password":"123"}|
|POST |	/cloud/logout  |Выход	         |Заголовок: auth-token             ||

### Работа с файлами

|Метод	|URL                            |Описание	    |Заголовки                       |
|-------|-------------------------------|---------------|--------------------------------|
|GET	|/cloud/list?limit=3	        |Список файлов  |auth-token                      |
|POST	|/cloud/file?filename=file.txt  |Загрузить файл |auth-token + файл               |
|GET	|/cloud/file?filename=file.txt  |Скачать файл   |auth-token                      |
|PUT	|/cloud/file?filename=old.txt   |Переименовать  |auth-token + {"name":"new.txt"} |
|DELETE	|/cloud/file?filename=file.txt  |Удалить файл   |auth-token                      |


## Локальный запуск приложения
* Склонировать репозиторий [https://github.com/IVoloshin89/CloudDiplom.git]
* Собрать проект
* Запустить приложение. Приложение будет запущено по адресу http://localhost:8080/cloud
* Необходимо перейти в консоль H2 database и добавить пользователей для авторизации в системе

## Запуск в Docker
* Собрать образ Docker docker build -t webcloud-app .
* Запустить контейнер на порту 8080
